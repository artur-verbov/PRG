Run instructions:
1) Create a new WinForms (.NET 6/7) project.
2) Replace Program.cs with the provided Program.cs.
3) Install NuGet packages: System.Data.SQLite and QRCoder.
4) Copy tickets.db into the project output folder (same folder as .exe) or the project's bin/Debug/net6.0-windows folder.
5) Run the application.
